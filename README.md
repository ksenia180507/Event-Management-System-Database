# Event Management System - Database Project

## Overview
This repository contains the SQL implementation and documentation for a comprehensive, 3NF-normalized relational database designed for an Event Management System.

The database efficiently manages structured data for real-world event planning, handling user roles, venues, ticketing, and session scheduling.

## Repository Contents
* `table_creationDB.sql`: DDL scripts containing the schema definition, including all tables, primary keys, and foreign key constraints.
* `data_insertionDB.sql`: DML scripts containing sample data to populate the database and simulate a real-world environment.
* `SQL_queries_eventsDB.sql`: A suite of analytical queries demonstrating CRUD operations, complex joins, aggregations, and transaction management.
* `Project_report_sql.pdf`: The complete project documentation, including the Entity Relationship Diagram (ERD) and detailed methodology.
* `ERD_event_management.png`: The Entity Relationship Disgram, which is made to show relationships between tables, their priimary and foreignn keys, and attributes.
